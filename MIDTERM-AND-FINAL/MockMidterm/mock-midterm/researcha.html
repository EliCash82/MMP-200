Bootstrap Documentation

I found during my 'Survey of the Field' that many people interested in the style of art (postmodernism) that I am planning to display visit the bootstrap documentation website more regularly than your average art critic. Upon inspection I found the Bootstrap Documentation page to have a very aesthetically pleasing navigation menu. Unlike the Bootstrap Navbar we've turned to in our class lab sessions, this side navigation can be toggled on or off from the left side of the screen.

When trying to find boilerplate code to create something similar to what I found in my UX/UI research I came across a github repository containing a code snippet titled `startbootstrap-simple-sidebar` which I injected into my final project folder. See images below.
